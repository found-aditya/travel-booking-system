# Travel Booking System

Built using Java, Servlets, JDBC, Spring, and MySQL.

## Demo Preview

Open [index.html](./index.html) to view the sample project preview in a browser.

Demo pages included:

- `index.html` for the home page
- `login.html` for the login screen
- `booking.html` for the reservation flow
- `dashboard.html` for the user dashboard

## Features

- User authentication
- Booking system
- MVC architecture
- MySQL database integration
- Secure session handling

## Description

This project is an online travel reservation system where users can log in, search travel options, and make bookings through a web-based application.

## Note

This repository currently contains a simple front-end demo preview to present the project structure and workflow clearly on GitHub. It can be expanded later with the full Java backend and database integration.

## Tech Stack

- Java
- Servlets
- JDBC
- Spring
- MySQL

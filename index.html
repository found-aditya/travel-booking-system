Travel Booking System

This is a simple front-end preview for an Online Travel Reservation System built using Java, Servlets, JDBC, Spring, and MySQL. It is meant to show the expected flow of the project on GitHub in a clear and practical way.

Book a Trip Open Login Page

Main Features

  • User registration and login
  • Search and booking workflow
  • Reservation management

Backend Design

  • MVC architecture using Servlets and Spring
  • Database connectivity with JDBC
  • Session-based authentication

Tech Stack

  • Java
  • Servlets
  • JDBC
  • Spring
  • MySQL
